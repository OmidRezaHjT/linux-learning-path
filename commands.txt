# Commands

## man 
"man [command]"
English: **displays the manual page for a command, showing usage, options, and descriptio**
**فارسی: **این دستور صفحه راهنمای یک دستور را نمایش می‌دهد و شامل نحوه استفاده، گزینه‌ها و توضیحات آن است

## ls
"ls [options]"
**English:** List files and directories in the current folder.  
**فارسی:** لیست گرفتن از فایل‌ها و دایرکتوری‌های موجود در پوشه فعلی.

## lsusb
"lsusb [options]"
English: **List all USB devices connected to the system**
**فارسی: **لیست تمام دستگاه‌های USB متصل به سیستم

## lspci
"lspci [options]"
English: **List all PCI devices (like graphics cards, network cards)**
**فارسی: **لیست تمام دستگاه‌های PCI (مثل کارت گرافیک، کارت شبکه)

## lsblk
"lsblk [options]"
English: **List information about all block devices (disks, partitions)**
**فارسی: **نمایش اطلاعات همه دستگاه‌های بلاک (هارد، پارتیشن‌ها)

## lshw
"lshw [options]"
English: **Display detailed hardware information of the system**
**فارسی: **نمایش اطلاعات سخت‌افزاری دقیق سیستم

## lsmod
"lsmod"
English: **Shows kernel modules. They are located at /lib/modules**
**فارسی: **نشان دادن ماژول های کرنل

## cat
"cat [file]"
English: **Display the content of a file**
**فارسی: **نمایش محتوای یک فایل

## mkdir
"mkdir [directory]"
English: **Create a new directory**
**فارسی: **ایجاد یک پوشه جدید

## rmdir
"rmdir [directory]"
English: **Remove an empty directory**
**فارسی: **حذف یک پوشه خالی

## rm
"rm [file/directory]"
English: **Remove a file or directory**
**فارسی: **حذف یک فایل یا پوشه

## touch
"touch [file]"
English: **Create a new empty file or update its timestamp**
**فارسی: **ایجاد یک فایل خالی یا به‌روزرسانی زمان آخرین تغییر آن

## cp
"cp [source] [destination]"
English: **Copy a file or directory**
**فارسی: **کپی کردن یک فایل یا پوش

## mv
"mv [source] [destination]"
English: **Move or rename a file or directory**
**فارسی: **جابجا کردن یا تغییر نام یک فایل یا پوشه

## pwd
"pwd"
English: **Show the current working directory**
**فارسی: **نمایش مسیر پوشه فعلی

## echo
"echo [text]"
English: **Print text to the terminal or write to a file**
**فارسی: **چاپ متن در ترمینال یا نوشتن در فایل

## insmod
"insmod [module]"
English: **Insert a kernel module into the system**
**فارسی: **اضافه کردن یک ماژول کرنل به سیستم

## modprobe
"modprobe [module]"
(Better option)
English: **Insert or remove a kernel module, handling dependencies automatically**
**فارسی: **اضافه یا حذف یک ماژول کرنل با مدیریت خودکار وابستگی‌ها

## rmmod
"rmmod [module]"
English: **Remove a kernel module from the system**
**فارسی: **حذف یک ماژول کرنل از سیستم

## modinfo
"modinfo [module]"
English: **Show information about a kernel module (version, dependencies, author, etc.)**
**فارسی: **نمایش اطلاعات درباره یک ماژول کرنل (نسخه، وابستگی‌ها، نویسنده و …)

## dmesg
"dmesg [options]"
English: **dmesg shows kernel messages but does not keep the boot process log permanently; you can use journalctl -b to view boot logs.**
**فارسی: **دستور dmesg پیام‌های کرنل را نشان می‌دهد اما لاگ پروسس بوت را نگه نمی‌دارد؛ برای دیدن لاگ بوت می‌توان از journalctl -b استفاده کرد

## pstree
"pstree [options]"
English: **pstree shows running processes in a tree-like structure, displaying parent-child relationships**
**فارسی: **این دستور فرآیندهای در حال اجرا را به صورت درختی نشان می‌دهد و رابطه والد و فرزند بین آن‌ها را نمایش می‌دهد

## systemctl
"systemctl [command]"
English: **systemctl is the main command to manage services in systemd (start, stop, enable, disable, check status)**
**فارسی: **دستور systemctl فرمان اصلی برای مدیریت سرویس‌ها در systemd است (اجرا، توقف، فعال‌سازی، غیرفعال‌سازی، بررسی وضعیت)

## systemctl list-units 
"systemctl list-units [options]"
English: **Lists all active units (services, sockets, targets, etc.) managed by systemd**
**فارسی: **لیست تمام یونیت‌های فعال (سرویس‌ها، سوکت‌ها، تارگت‌ها و …) که توسط systemd مدیریت می‌شوند

## systemctl get-default
"systemctl get-default"
English: **Shows the default target, i.e., which group of services the system starts by default (like multi-user or graphical)**
**فارسی: **تارگت پیش‌فرض سیستم را نشان می‌دهد؛ یعنی گروهی از سرویس‌ها که سیستم به طور پیش‌فرض اجرا می‌کند (مثلاً multi-user یا graphical)

## systemctl is-system-[state]
"systemctl is-system-[state]"
English: **checks the overall state of the system. You replace [state] with one option at a time (running, degraded, maintenance, initializing, starting, stopping) to see if the system is in that specific state**
**فارسی: **این دستور وضعیت کلی سیستم را بررسی می‌کند. [state] را هر بار با یکی از گزینه‌ها (running، degraded، maintenance، initializing، starting، stopping) جایگزین می‌کنید تا مشخص شود سیستم در آن وضعیت خاص است یا خیر

## journalctl
"journalctl [options]"
English: **journalctl shows logs collected by systemd’s journal, including system, kernel, service, and boot logs (e.g., journalctl -k for kernel, journalctl -b for boot)**
**فارسی: **دستور journalctl لاگ‌های جمع‌آوری‌شده توسط journal در systemd را نمایش می‌دهد، از جمله لاگ‌های سیستم، کرنل، سرویس‌ها و بوت (مثلاً journalctl -k برای کرنل و journalctl -b برای بوت)

## runlevel
"runlevel"
English: **you can check your current runlevel with this command/ The default was in /etc/inittab **
**فارسی: ** می‌تونی وضعیت ران لول فعلی خودت را با این دستور بررسی کنی

## telinit
"telinit [runlevel]"
English: **you can changes the current runlevel of a SysVinit system, effectively switching the system to a different stage of operation.**
**فارسی: **دستور telinit runlevel فعلی سیستم SysVinit را تغییر می‌دهد و سیستم را به مرحله‌ای دیگر از عملکرد منتقل می‌کند.
Note: /etc/inittab is used by SysVinit. systemd does not use this file ; it uses targets instead (e.g., default.target) 

## shutdown
"shutdown [-r | -h | -P] [time] [message]"
English: **shutdown safely powers off or reboots a Linux system, optionally notifying users; it can also perform a halt (stop all processes and CPU) if specified(e.g., shutdown -r 60 Reloading updated kernel)
**
**فارسی: **دستور shutdown برای خاموش یا ریبوت امن یک سیستم لینوکس استفاده می‌شود و می‌توان کاربران را مطلع کرد؛ همچنین می‌تواند halt (توقف تمام پردازش‌ها و CPU) را نیز انجام دهد

## wall
"wall [message]"
English: **Sends a message to all logged-in users on the system**
**فارسی: **این دستور پیامی را به تمام کاربران وارد شده در سیستم ارسال می‌کند

## mesg
"mesg [y|n]"
English: **Controls whether other users can write to your terminal (allow y or deny n)**
**فارسی: **این دستور مشخص می‌کند که آیا کاربران دیگر می‌توانند به ترمینال شما پیام بفرستند

## /etc/motd
English: **The “Message of the Day” file, displayed to users after they log in to the system**
**فارسی: **فایل “پیام روز” که پس از ورود کاربران به سیستم نمایش داده می‌شود

## /etc/issue
English: **A text file displayed before the login prompt, often used to show system information or legal notices**
**فارسی: **یک فایل متنی که قبل از نمایش صفحه ورود نشان داده می‌شود و معمولاً برای نمایش اطلاعات سیستم یا هشدارهای قانونی استفاده می‌شود
