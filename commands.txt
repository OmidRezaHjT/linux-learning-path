## man

English: **Displays the manual page for a command, showing usage, options, and description**

## systemctl get-default

## LVM
"lvm [command]"
English: **Logical Volume Manager: flexible disk management using PV, VG, LV

PV (Physical Volume): A whole disk or a partition. It’s better to use partitions rather than raw disks.
VG (Volume Group): A collection of one or more PVs. The OS sees it as one big disk. PVs can differ in size or be on different physical disks.
LV (Logical Volume): Appears as a partition to the OS. It can be formatted and used like a regular partition.
**

Finglish: **Modiriat-e mojaz-e volume dar Linux, estefade az PV, VG, LV baraye modiriat-e fleksibel-e disk**

English: **Used to manage packages in Debian-based systems (like Ubuntu). It can install, update, upgrade, search, and remove software (update: refreshes package lists, upgrade: installs newer versions of installed packages)(Important switches/commands: install = install package, remove = remove package, update = update package lists, upgrade = upgrade all packages, search = search for package, show = show package info)**

## apt-get
"apt-get [options] [command]"
English: **Used for package management in Debian-based systems (older tool than apt). It can install, update, upgrade, and remove packages (update: refreshes package lists, upgrade: installs newer versions of installed packages)(Important switches/commands: install = install package, remove = remove package, update = update package lists, upgrade = upgrade packages, dist-upgrade = smart upgrade handling dependencies, autoremove = remove unused packages)**
Finglish: **Baraye modiriyat package dar system-haye Debian estefade mishe (ghabl az apt bood). Mitune package-ha ro nasb, update, upgrade ya hazf kone (update: list package-ha ro tazeh mikone, upgrade: version jadid package-haye nasb shode ro nasb mikone)(Important switches/commands: install = install package, remove = remove package, update = update package lists, upgrade = upgrade packages, dist-upgrade = smart upgrade handling dependencies, autoremove = remove unused packages)**

English: Command-line tool to query and search the APT package cache. Komak mikone package-ha, dependency-ha va joz’iyat-e package ro peyda koni vali nasb ya hazf nemikone (Important switches: search = search package, show = show details, depends = list dependencies, rdepends = list reverse dependencies, policy = show install candidate)

## dpkg
"dpkg [options] [command]"
English: **Low-level package manager for Debian-based systems. It installs, removes, and provides information about .deb packages but does not handle dependencies automatically (Important switches: -i = install, -r = remove, -l = list, -s = status, -L = show files in package)**
Finglish: **Package manager dar sath-e payin baraye Debian-base shode. Baraye nasb, hazf va daryaft etela'at dar mored .deb package-ha estefade mishe, vali dependency-ha ro khodesh automatic handle nemikone(Important switches: -i = install, -r = remove, -l = list, -s = status, -L = show files in package)** 
  In some cases a package is installed but without proper dependencies (say using dpkg) or an installation is interrupted for any reason. In these cases a apt-get install -f might help, -f is for fix broken :)

## yum
"yum [option] [command]"
English: **Command-line package manager for RPM-based distributions (like CentOS, RHEL, Fedora before DNF), handles dependencies automatically(Important switches/commands:install = install package, remove = remove package, update = update installed packages, search = search for packages, info = show package info, list = list available/installed packages,clean all = clear cache and metadata)
yum update → Updates all packages to the latest version, but does not remove obsolete packages
yum upgrade → Updates all packages and removes obsolete or unnecessary ones***
Finglish: **Abzar modiriyat pakesh baraye distro-haye RPM (mesl CentOS, RHEL, Fedora ghabl az DNF), ba tamarkoz roye hal kardan dependency-ha(install = install package, remove = remove package, update = update installed packages, search = search for packages, info = show package info, list = list available/installed packages,clean all = clear cache and metadata)
yum update → hame package-ha ro be latest version update mikone, vali obsolete-ha ro pak nemikone
yum upgrade → mesle update hast vali obsolete o extra package-ha ro ham remove mikone**


## rpm
"rpm [option] [command]"
English: **Low-level package manager for Red Hat-based systems, works directly with .rpm files (not handling dependencies automatically)(Important switches/commands: -i = install, -U = upgrade, -e = erase/remove, -q = query package, -V = verify package integrity, -qa = list all installed packages)**
Finglish: **Modir pakesh dar Red Hat-based ke mostaghim ba file-haye .rpm kar mikone (be sorat otomatik dependensi-ha ro hal nemikone)(Important switches/commands: -i = install, -U = upgrade, -e = erase/remove, -q = query package, -V = verify package integrity, -qa = list all installed packages)**

## dnf
"dnf [options] [command]"
English: **Modern package manager for RPM-based systems (replacement for yum). It can install, update, upgrade, and remove packages (Important switches/commands: update: refreshes package lists and updates packages, upgrade: same as update but also handles obsolete packages)(Important switches/commands: install = install package, remove = remove package, update = update packages, upgrade = update + handle obsolete packages, autoremove = remove unused dependencies, search = search packages, info = show package information)**
Finglish: Package manager jadid baraye system-haye RPM-based (jaygozin yum). Mitune package-ha ro nasb, update, upgrade va hazf kone (Important switches/commands: update: list va package-ha ro update mikone, upgrade: mesle update vali obsolete package-ha ro ham handle mikone)(Important switches/commands: install = install package, remove = remove package, update = update packages, upgrade = update + handle obsolete packages, autoremove = remove unused dependencies, search = search packages, info = show package information)**
